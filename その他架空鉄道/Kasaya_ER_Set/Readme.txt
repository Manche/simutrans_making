このセットには以下の車輌が含まれています。
フコ100形・モコ500形
フコ260形・モコ550形
タモ300形
1932年製造、2003年全廃
コは制御車、モはモーター車、フは付随車、タは単行車
いずれも開業時に製造された。
フコ100形とモコ500形は15m級の車体を持ち、2ドアを有していた。
主に賀佐谷〜金華間で用いられ、閑散期には賀佐谷〜広兼間でも運用されていた。
ラッシュ時にはフコ260形・モコ550形を増結する時がある。
フコ260形とモコ550形は20m級の車体を持ち、4ドアを有していた。
主に賀佐谷〜広兼間でのみ運用されていた。車体が重く、最高速度も低いため、フコ100形やタモ300形よりも早く廃車された。
タモ300形は15m級の車体を持ち、2ドアを有していた。
主に賀佐谷〜城戸間で運用され、稀にフコ100形やモコ500形と併結し、賀佐谷〜金華間でも運用されたことがある。
いずれも後継となる1000系の登場により、ほとんどが1978年に全廃されたが、タモ300形のみ路線廃止の2003年まで運行された。

1000系
1961年製造、2007年全廃
在来車置き換え用に登場した。
103系と同様の車体を持ち、最高速度は110km/hまで向上した。(路線そのものの最高速度向上は1500系による1978年の本線在来車淘汰後）
一時期はタモ300形と併結し、賀佐谷〜城戸間での運用を行っていたが、1993年には本線のみの運用となった。
2000系登場の2005年には1編成を除いて全て廃車、残る1編成も2500系登場により、廃車された。

1500系
1977年製造、2010年全廃
1000系増備車として登場した。
この形式の登場により、在来車が淘汰され、路線最高速度が向上した。
ほぼ1000系そのままではあるが、登場時期が異なるため、形式が別となった。
2500系量産3編成目が落成したその年に大半が廃車、残る3編成も2500系増備車の登場後に廃車された。

1700系
1986年製造、2005年体質改善工事実施
暮方鉄道暮方工場から5編成製造された。
3ドアを有しており、クロスシートが備わっている。
主に快速などの優等運用に使用され、本線全線で運用されている。
2005年に体質改善工事が行われ、最高速度の向上がなされた。

2000系
1995年製造
1500系の後継として登場した。
ステンレス車体を持ち、4ドアを有している。

2100系
1997年製造
2000系の増備車として登場した。
暮方鉄道1000系をベースにしている。

2500系
2009年製造
JR西日本の225系をベースとして登場した。
この車両の登場により、在籍車が全てVVVF車となった。
優等用車両としてこの車両をベースとした2600系が2011年に登場している。
