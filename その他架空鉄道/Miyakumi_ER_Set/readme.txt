----
[宮汲急行電鉄セット]


----
宮汲急行電鉄セットのご利用ありがとうございます。

130km/hで快走していく青海電車とはまた違ったローカル線っぽい車両のセットです。

概要：
開業は比較的遅く、1963年と青海電車新線の開業後となっている。
青海電車西久谷駅から本線は白沼駅へ、河海線は河海港へ、
さらに湖南線は昭東電鉄へ連絡する城山駅へ走っている。

車両：
100系
運用開始1963年
開業時から運用されている車両で、青海電車宮津工機から5編成製造された。
モ100とモ500の二両を1ユニットとしている。
登場時は車体全面が若草色の塗装となっていたが、1983年の冷房化後は赤緑の帯のみの塗装となっている。

200系
運用開始1973年
モ200とモ600の二両を1ユニットとしている。
登場時は車体下部が若草色、赤緑の帯となっていた。
1983年の冷房化後は塗色が変更され、赤緑帯のみのシンプルな塗装となった。

300系
運用開始1977年
増加する乗客に対応するため、宮汲急行初の4両固定編成となった。
Mc車がモ300、M車がモ400/450、Tc車がモ700となっている。
登場時から冷房化され、なおかつ先頭部が鋼製、それ以外がステンレス製となっている。

500系
運用開始1984年
