新川電車リニューアルセットのご利用ありがとうございます。

以前製作したリニューアル品となります。
ほぼ新規製作と同じのため、以前に公開していたものは削除推奨です。
家具産業セット、青海電車S1000系は非同梱です。

555氏の阪神電車セット
から改造させていただきました。

この場を借りて感謝申し上げます。
----------------
このセットには以下の車輌が含まれています。
・新川電車
300系
1951年製造、1998年全廃
新川〜倉川間開業につき製造された。
制御方式は抵抗制御、ブレーキはHSC電磁直通ブレーキを使用する。
製造当初から冷房が設置され、路線の有名化に一役買った。
晩年には後継車種に負けじと快速運用に入る等、第一線で長らく活躍していた。

500系
1953年製造、2010年全廃
寺上鉄道直通に伴い増発用として製造。
車体は300系と一部同じだが、2扉でモノコック構造となっている。

550系
1958年製造、1962年全廃
500系と同様の構造だが、こちらは雨樋が存在する。
湾岸線に一時転用されたが、再び本線に戻った。
1000系の登場と500系への改造により4年という短い期間で形式消滅。

1000系
1964年製造、1980年全廃
湾岸線ローカル区間用として製造した。
湾岸線が本線との一体型運用へと計画が変更されたため、主に増結車として運用に入った。
本線8連化に伴い空港線へと転用されたものの、本線のダイヤ乱れによっては新川まで足を運ぶことがあった。
6000系やその短編成版の6050系の登場により全廃。

2000系
1970年製造、2005年全廃
500系や550系の置き換え用、東荘原暮之宮へ終点が移ったことによる輸送体系の変化に対応するため製造。
1000系・500系の設計を流用しており、両形式を足して割った雰囲気を持つ。

3000系
1973年製造、2008年全廃
2000系の仕様変更車として冷房をより強いものへ変更した。

5000系
1975年製造、2009年全廃
輸送力の増強と在来形式の代替のため製造。
基本設計はそのままに扉の数を増やしたほか、一部編成には試験的に界磁チョッパ制御とされている。

5500系
1979年製造
5000系の近代化車体改造車。
5000系の中で界磁チョッパ制御の編成には車体改良工事と耐腐食工事が行われた。

6000系
1992年製造
湾岸線改造計画として耐腐食性に優れたオールステンレス性の車体を持つ車両として製造。
バーニア制御による多段方式を採用し、起動時の衝撃を減らすように工夫されている。
主制御器の制御段数は95段、ブレーキは85段となっており、再加速時のタイムラグを減らすため速度に応じて抵抗器の段数が自動で変化される機構を備えている。

6050系
1996年一部改造
6000系の短編成化バージョン

8000系
2011年製造
5500系の更新工事の予備車の確保と代替を目的に製造。
